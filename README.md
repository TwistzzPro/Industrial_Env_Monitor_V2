# Industrial Environment Monitor V2

## Project Overview

基于 STM32F407 的工业环境监测终端。

主要功能：

- 温湿度采集（SHT30）
- 光照采集（BH1750）
- FreeRTOS多任务管理
- RS485通信
- Modbus RTU协议
- Git版本管理
- 逻辑分析仪调试

## Hardware

- STM32F407
- SHT30
- BH1750
- MAX485

## Software

- STM32CubeMX
- Keil MDK
- FreeRTOS
